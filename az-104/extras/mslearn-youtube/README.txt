These are extracts based on the Microsoft Learn YouTube channel, which provides a variety of videos on Azure and other Microsoft technologies.
The content is designed to help learners understand and master Azure concepts, services, and best practices.

The flow for creating these markdowns is :


Get the transcript from the youtube video

Uses the following scripts from https://github.com/ddewaele/podcast-insighter/tree/main/scripts

```
python fetch_transcript.py https://www.youtube.com/watch\?v\=Zxb6igaFheU
python analyze.py output/Zxb6igaFheU/transcript.txt
```
