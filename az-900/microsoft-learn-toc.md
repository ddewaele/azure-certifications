# AZ-900 Microsoft Learn Course Syllabus

Source: [Course AZ-900T00-A — Introduction to Cloud Infrastructure](https://learn.microsoft.com/en-us/training/courses/az-900t00)

The course is structured as three learning paths, each made up of several modules. Each module contains a set of units.

---
