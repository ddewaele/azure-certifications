# Microsoft 365 Admin Centers Quick Reference

## Which Admin Center for What?

| Task | Admin Center | URL |
|------|-------------|-----|
| Create/manage users and groups | M365 admin center | admin.microsoft.com |
| Assign licenses | M365 admin center | admin.microsoft.com |
| Manage org settings and domains | M365 admin center | admin.microsoft.com |
| View service health | M365 admin center | admin.microsoft.com |
| Manage Copilot settings | M365 admin center | admin.microsoft.com |
| Manage mailboxes | Exchange admin center | admin.exchange.microsoft.com |
| Manage distribution lists | Exchange admin center | admin.exchange.microsoft.com |
| Configure mail flow rules | Exchange admin center | admin.exchange.microsoft.com |
| Manage SharePoint sites | SharePoint admin center | admin.microsoft.com/sharepoint |
| Configure sharing policies | SharePoint admin center | admin.microsoft.com/sharepoint |
| Run data access governance reports | SharePoint admin center | admin.microsoft.com/sharepoint |
| Manage teams and channels | Teams admin center | admin.teams.microsoft.com |
| Configure meeting/messaging policies | Teams admin center | admin.teams.microsoft.com |
| Manage app permissions | Teams admin center | admin.teams.microsoft.com |
| Manage users, groups, Conditional Access | Entra admin center | entra.microsoft.com |
| Review Identity Secure Score | Entra admin center | entra.microsoft.com |
| Investigate sign-in issues | Entra admin center | entra.microsoft.com |
| Configure PIM | Entra admin center | entra.microsoft.com |
| Manage sensitivity labels, DLP | Purview portal | purview.microsoft.com |
| Review Compliance Manager | Purview portal | purview.microsoft.com |
| Manage retention policies | Purview portal | purview.microsoft.com |
| Investigate insider risks | Purview portal | purview.microsoft.com |
| Manage Copilot Studio agents | Power Platform admin center | admin.powerplatform.microsoft.com |
| Agent approval workflows | Power Platform admin center | admin.powerplatform.microsoft.com |
| Agent operational insights | Power Platform admin center | admin.powerplatform.microsoft.com |
| Threat detection and response | Microsoft Defender portal | security.microsoft.com |
