# AB-900 -- Microsoft 365 Copilot and Agent Administration Fundamentals

Study guide and practice materials for the **Microsoft 365 Certified: Copilot and Agent Administration Fundamentals** certification.

## Exam Overview

| Detail | Value |
|--------|-------|
| Exam code | AB-900 |
| Level | Fundamentals |
| Duration | 45 minutes |
| Passing score | 700 / 1000 |
| Question types | Multiple choice, multiple select |
| Cost | ~$99 USD (varies by country) |
| Prerequisites | Familiarity with M365, security, and Copilot concepts |
| Languages | English |

## Exam Domains

| # | Domain | Weight |
|---|--------|--------|
| 1 | Identify the core features and objects of Microsoft 365 services | 30-35% |
| 2 | Understand data protection and governance tasks for Microsoft 365 and Copilot | 35-40% |
| 3 | Perform basic administrative tasks for Copilot and agents | 25-30% |
