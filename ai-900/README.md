# AI-900 -- Azure AI Fundamentals

Study guide and practice materials for the **Microsoft Certified: Azure AI Fundamentals** certification.

> **Note:** The exam code is AI-900. The course AI-901T00 (Introduction to AI in Azure) replaces the older AI-900T00 course but still prepares for the AI-900 exam.

> **Important:** This certification retires on **June 30, 2026**.

## Exam Overview

| Detail | Value |
|--------|-------|
| Exam code | AI-900 |
| Course code | AI-901T00 |
| Duration | 45 minutes |
| Passing score | 700 / 1000 |
| Question types | Multiple choice, multiple select, drag-and-drop, case studies |
| Cost | ~$99 USD (varies by country) |
| Prerequisite | None (basic cloud and math awareness recommended) |

## Exam Domains

| # | Domain | Weight |
|---|--------|--------|
| 1 | Describe Artificial Intelligence workloads and considerations | 15-20% |
| 2 | Describe fundamental principles of machine learning on Azure | 15-20% |
| 3 | Describe features of computer vision workloads on Azure | 15-20% |
| 4 | Describe features of Natural Language Processing (NLP) workloads on Azure | 15-20% |
| 5 | Describe features of generative AI workloads on Azure | 20-25% |
