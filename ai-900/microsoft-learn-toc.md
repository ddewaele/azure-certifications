# AI-900 Microsoft Learn Course Syllabus

Course **AI-901T00: Introduction to AI in Azure** prepares for exam **AI-900: Azure AI Fundamentals**.

> This course replaces the older AI-900T00 course. The exam code remains AI-900.

---

## Learning Path 1: AI Concepts for Developers and Technology Professionals

Covers the conceptual foundations of AI, generative AI, NLP, speech, computer vision, and information extraction.

| # | Module | Exam Domain |
|---|--------|-------------|
| 1 | Introduction to AI concepts | Domain 1: AI workloads and considerations |
| 2 | Introduction to generative AI and agents | Domain 5: Generative AI |
| 3 | Introduction to natural language processing concepts | Domain 4: NLP |
| 4 | Introduction to AI speech concepts | Domain 4: NLP |
| 5 | Introduction to computer vision concepts | Domain 3: Computer vision |
| 6 | Introduction to AI-powered information extraction concepts | Domain 1: AI workloads (document processing) |

**Link:** https://learn.microsoft.com/en-us/training/paths/ai-concepts/

---

## Learning Path 2: Get Started with AI Applications and Agents on Azure

Hands-on modules using Microsoft Foundry (formerly Azure AI Studio) to build and test AI solutions.

| # | Module | Exam Domain |
|---|--------|-------------|
| 1 | Get started with AI in Azure | Domain 1: AI workloads and considerations |
| 2 | Get started with generative AI and agents in Azure | Domain 5: Generative AI |
| 3 | Get started with text analysis in Azure | Domain 4: NLP |
| 4 | Get started with speech in Azure | Domain 4: NLP |
| 5 | Get started with computer vision in Azure | Domain 3: Computer vision |
| 6 | Get started with AI-powered information extraction in Azure | Domain 1: AI workloads (document processing) |

**Link:** https://learn.microsoft.com/en-us/training/paths/get-started-ai-apps-agents/

---

## Exam Domain Coverage Map

| Exam Domain | Weight | Learning Path 1 Modules | Learning Path 2 Modules |
|-------------|--------|------------------------|------------------------|
| 1. AI workloads and considerations | 15-20% | 1, 6 | 1, 6 |
| 2. Machine learning fundamentals | 15-20% | 1 (partial) | 1 (partial) |
| 3. Computer vision | 15-20% | 5 | 5 |
| 4. NLP | 15-20% | 3, 4 | 3, 4 |
| 5. Generative AI | 20-25% | 2 | 2 |

> **Note:** Domain 2 (Machine Learning fundamentals) has limited dedicated coverage in the AI-901 course modules. The concepts are partially covered in Module 1 of each learning path. Supplementary study using the [ML fundamentals concept guide](./concepts/02-machine-learning-fundamentals.md) is recommended.
