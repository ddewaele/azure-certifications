# AI-102 Microsoft Learn Course Syllabus

Exam **AI-102: Designing and Implementing a Microsoft Azure AI Solution** certifies you as a **Microsoft Certified: Azure AI Engineer Associate**.

> This certification retires on **June 30, 2026**.

---

## Exam Domains and Weights

| # | Domain | Weight |
|---|--------|--------|
| 1 | Plan and manage an Azure AI solution | 20-25% |
| 2 | Implement generative AI solutions | 15-20% |
| 3 | Implement an agentic solution | 5-10% |
| 4 | Implement computer vision solutions | 10-15% |
| 5 | Implement natural language processing solutions | 15-20% |
| 6 | Implement knowledge mining and information extraction solutions | 15-20% |

---

## Recommended Learning Paths

### Domain 1: Plan and Manage an Azure AI Solution

| Module | Topics |
|--------|--------|
| Get started with Azure AI Services | Resource creation, multi-service vs single-service, endpoints, keys, SDKs |
| Secure Azure AI Services | Key management, Key Vault, managed identity, Entra ID auth, RBAC roles |
| Monitor Azure AI Services | Azure Monitor, diagnostic settings, Application Insights, cost management |
| Deploy Azure AI in containers | Container images, configuration, billing, offline scenarios |
| Implement responsible AI | Content Safety, content filters, blocklists, prompt shields, governance |

### Domain 2: Implement Generative AI Solutions

| Module | Topics |
|--------|--------|
| Get started with Azure OpenAI | Resource provisioning, model deployment, Chat Completions API |
| Build apps with Azure OpenAI | SDK integration, prompt engineering, system messages, parameters |
| Implement RAG with Azure AI Search | Vector indexing, hybrid search, chunking, embeddings, prompt augmentation |
| Work with Microsoft Foundry | Hubs, projects, model catalog, prompt flow, evaluation |
| Optimize generative AI | Fine-tuning, monitoring, container deployment, prompt engineering |

### Domain 3: Implement an Agentic Solution

| Module | Topics |
|--------|--------|
| Build AI agents | Agent Service, threads, runs, tools (code interpreter, file search, function calling) |
| Multi-agent solutions | Orchestrator pattern, Semantic Kernel, multi-agent workflows |

### Domain 4: Implement Computer Vision Solutions

| Module | Topics |
|--------|--------|
| Analyse images with Azure AI Vision | Image Analysis 4.0, captions, tags, objects, OCR, people detection |
| Custom vision models | Image classification, object detection, training, evaluation, publishing |
| Analyse video | Video Indexer insights, Spatial Analysis for people movement |

### Domain 5: Implement NLP Solutions

| Module | Topics |
|--------|--------|
| Analyse text with Azure AI Language | Sentiment, NER, key phrases, PII, entity linking, summarisation |
| Build speech solutions | STT, TTS, SSML, speech translation, custom speech, speaker recognition |
| Build conversational language understanding | Intents, entities, utterances, training, deployment |
| Create question answering solutions | Knowledge base, alternate phrasings, multi-turn, chit-chat |
| Translate text and speech | Translator API, document translation, custom translator |

### Domain 6: Implement Knowledge Mining and Information Extraction

| Module | Topics |
|--------|--------|
| Create an Azure AI Search solution | Index, indexer, data source, skillset, knowledge store |
| Create custom skills | WebApiSkill, Azure Functions, input/output schemas |
| Implement vector and semantic search | Embeddings, HNSW, hybrid search, semantic ranker |
| Extract data with Document Intelligence | Prebuilt models, custom models, composed models |
| Azure Content Understanding | OCR pipelines, document classification, entity/table extraction |
